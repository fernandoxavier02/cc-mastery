# CC-Toolkit

Claude Code Swiss Army Knife — diagnose, configure, and build your Claude Code environment.

## Installation

```bash
claude plugin add https://github.com/fernandobt/cc-toolkit
```

## Skills

| Command | Description |
|---------|-------------|
| `/cc` | Interactive triage menu |
| `/cc-audit` | Audit project context files |
| `/cc-context` | Diagnose context problems |
| `/cc-mcp` | Configure MCP servers |
| `/cc-hooks` | Create and debug hooks |
| `/cc-agent` | Design agents and subagents |
| `/cc-prompt` | Optimize prompts and API usage |
| `/cc-security` | Permissions and security |
| `/cc-setup` | Setup and troubleshooting |
| `/cc-skill` | Create skills and commands |

## Quick Start

Type `/cc` to get an interactive menu, or invoke any skill directly.

## Customization

Override any skill by placing a modified `SKILL.md` in your project's `.claude/skills/` directory.
